## 📊 一、当前状态分析

### 1.1 现有实现

**前端登出逻辑**（`PortalLayout.tsx`）:
```typescript
const onLogout = () => {
  clearToken();  // 只清除localStorage
  navigate('/login');
};
```

**问题**：
- ❌ 没有通知后端用户已登出
- ❌ JWT Token仍然有效（直到过期）
- ❌ 无法清除服务端状态（在线状态、Redis缓存等）
- ❌ 无法实现"强制下线"功能
- ❌ 无法统计登出日志

---

## 🎯 二、方案对比

### 2.1 方案A：纯前端登出（当前方案）

**实现**：只清除前端localStorage

**优点**：
- ✅ 简单快速
- ✅ 无需后端支持

**缺点**：
- ❌ Token仍然有效（安全风险）
- ❌ 无法清除服务端状态
- ❌ 无法实现强制下线
- ❌ 不符合企业级安全标准

**适用场景**：仅用于开发/测试环境

---

### 2.2 方案B：后端登出接口 + Token黑名单（推荐）

**实现**：
1. 前端调用登出接口
2. 后端将Token加入黑名单（Redis）
3. 后端清除用户在线状态
4. JWT Filter检查黑名单

**优点**：
- ✅ 安全性高（Token立即失效）
- ✅ 可以清除服务端状态
- ✅ 支持强制下线
- ✅ 符合企业级标准

**缺点**：
- ⚠️ 需要Redis存储黑名单
- ⚠️ 每次请求需要检查黑名单（性能影响小）

**适用场景**：**生产环境推荐**

---

### 2.3 方案C：后端登出接口 + 短Token过期时间

**实现**：
1. 前端调用登出接口
2. 后端清除用户状态
3. Token依赖过期时间自然失效

**优点**：
- ✅ 实现简单
- ✅ 无需黑名单

**缺点**：
- ❌ Token在过期前仍然有效（安全风险）
- ❌ 无法立即失效Token

**适用场景**：对安全性要求不高的场景

---

### 3.2 实现步骤

#### 步骤1：创建Token黑名单服务

**文件**: `src/main/java/com/ityfz/yulu/common/security/TokenBlacklistService.java`

```java
package com.ityfz.yulu.common.security;

import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.data.redis.core.RedisTemplate;
import org.springframework.stereotype.Service;

import java.util.concurrent.TimeUnit;

/**
 * Token黑名单服务
 * 用于实现登出功能，使Token立即失效
 */
@Slf4j
@Service
@RequiredArgsConstructor
public class TokenBlacklistService {
    
    private final RedisTemplate<String, Object> redisTemplate;
    
    private static final String BLACKLIST_KEY_PREFIX = "token:blacklist:";
    
    /**
     * 默认Token过期时间（秒），用于设置黑名单TTL
     * 应该与JWT Token的过期时间一致
     */
    private static final long DEFAULT_TOKEN_EXPIRE_SECONDS = 7 * 24 * 3600; // 7天
    
    /**
     * 将Token加入黑名单
     * 
     * @param token JWT Token
     * @param expireSeconds 过期时间（秒），如果为null则使用默认值
     */
    public void addToBlacklist(String token, Long expireSeconds) {
        if (token == null || token.trim().isEmpty()) {
            return;
        }
        
        String key = BLACKLIST_KEY_PREFIX + token;
        long ttl = expireSeconds != null ? expireSeconds : DEFAULT_TOKEN_EXPIRE_SECONDS;
        
        redisTemplate.opsForValue().set(key, "1", ttl, TimeUnit.SECONDS);
        
        log.info("[TokenBlacklist] Token已加入黑名单，TTL={}秒", ttl);
    }
    
    /**
     * 检查Token是否在黑名单中
     * 
     * @param token JWT Token
     * @return true-在黑名单中，false-不在
     */
    public boolean isBlacklisted(String token) {
        if (token == null || token.trim().isEmpty()) {
            return false;
        }
        
        String key = BLACKLIST_KEY_PREFIX + token;
        Boolean exists = redisTemplate.hasKey(key);
        
        return Boolean.TRUE.equals(exists);
    }
    
    /**
     * 从黑名单移除Token（一般不需要，等TTL过期即可）
     * 
     * @param token JWT Token
     */
    public void removeFromBlacklist(String token) {
        if (token == null || token.trim().isEmpty()) {
            return;
        }
        
        String key = BLACKLIST_KEY_PREFIX + token;
        redisTemplate.delete(key);
        
        log.info("[TokenBlacklist] Token已从黑名单移除");
    }
}
```

#### 步骤2：修改JWT Filter检查黑名单

**文件**: `src/main/java/com/ityfz/yulu/common/security/JwtAuthFilter.java`

```java
// 在 doFilterInternal 方法中，验证Token后添加黑名单检查

@Override
protected void doFilterInternal(HttpServletRequest request, 
                                HttpServletResponse response, 
                                FilterChain filterChain) throws ServletException, IOException {
    
    String token = extractToken(request);
    
    if (token == null) {
        filterChain.doFilter(request, response);
        return;
    }
    
    try {
        // 1. 验证Token是否有效
        Claims claims = jwtUtil.parseToken(token);
        if (claims == null) {
            filterChain.doFilter(request, response);
            return;
        }
        
        // 2. 检查Token是否在黑名单中（新增）
        if (tokenBlacklistService.isBlacklisted(token)) {
            log.warn("[JWT] Token已被加入黑名单: {}", token.substring(0, 20) + "...");
            response.setStatus(HttpServletResponse.SC_UNAUTHORIZED);
            response.setContentType("application/json;charset=UTF-8");
            response.getWriter().write("{\"code\":\"401\",\"message\":\"Token已失效，请重新登录\"}");
            return;
        }
        
        // 3. 设置上下文（原有逻辑）
        Long userId = Long.parseLong(claims.getSubject());
        Long tenantId = claims.get("tenantId", Long.class);
        String role = claims.get("role", String.class);
        
        TenantContextHolder.setTenantId(tenantId);
        UserContextHolder.setUserId(userId);
        UserContextHolder.setRole(role);
        
        filterChain.doFilter(request, response);
        
    } catch (Exception e) {
        log.error("[JWT] Token验证失败", e);
        filterChain.doFilter(request, response);
    } finally {
        TenantContextHolder.clear();
        UserContextHolder.clear();
    }
}
```

#### 步骤3：创建统一登出Controller

**文件**: `src/main/java/com/ityfz/yulu/common/controller/AuthController.java` (新建)

```java
package com.ityfz.yulu.common.controller;

import com.ityfz.yulu.common.annotation.RequireRole;
import com.ityfz.yulu.common.enums.ErrorCodes;
import com.ityfz.yulu.common.enums.Roles;
import com.ityfz.yulu.common.exception.BizException;
import com.ityfz.yulu.common.model.ApiResponse;
import com.ityfz.yulu.common.security.JwtUtil;
import com.ityfz.yulu.common.security.SecurityUtil;
import com.ityfz.yulu.common.security.TokenBlacklistService;
import com.ityfz.yulu.user.service.AgentStatusService;
import io.jsonwebtoken.Claims;
import jakarta.servlet.http.HttpServletRequest;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RestController;

/**
 * 统一认证Controller（登录/登出）
 */
@Slf4j
@RestController
@RequestMapping("/api")
@RequiredArgsConstructor
public class AuthController {
    
    private final TokenBlacklistService tokenBlacklistService;
    private final AgentStatusService agentStatusService;
    private final JwtUtil jwtUtil;
    
    /**
     * 统一登出接口
     * POST /api/logout
     * 
     * 支持所有角色（USER/ADMIN/AGENT）
     */
    @PostMapping("/logout")
    public ApiResponse<Void> logout(HttpServletRequest request) {
        // 1. 获取当前用户信息
        Long tenantId = SecurityUtil.currentTenantId();
        Long userId = SecurityUtil.currentUserId();
        String role = SecurityUtil.currentRole();
        
        if (tenantId == null || userId == null) {
            // 如果没有登录信息，直接返回成功（避免重复登出报错）
            return ApiResponse.success("登出成功");
        }
        
        // 2. 提取Token
        String token = extractToken(request);
        
        // 3. 将Token加入黑名单
        if (token != null) {
            // 计算Token剩余有效时间，设置黑名单TTL
            Long expireSeconds = calculateTokenExpireSeconds(token);
            tokenBlacklistService.addToBlacklist(token, expireSeconds);
            log.info("[Logout] Token已加入黑名单: userId={}, tenantId={}", userId, tenantId);
        }
        
        // 4. 清除客服在线状态（如果是客服或管理员）
        if (Roles.isAgent(role) || Roles.isAdmin(role)) {
            try {
                agentStatusService.setOffline(tenantId, userId);
                log.info("[Logout] 客服状态已清除: userId={}, tenantId={}", userId, tenantId);
            } catch (Exception e) {
                log.warn("[Logout] 清除客服状态失败", e);
                // 不影响登出流程，继续执行
            }
        }
        
        // 5. 记录登出日志（可选）
        log.info("[Logout] 用户登出成功: userId={}, tenantId={}, role={}", userId, tenantId, role);
        
        return ApiResponse.success("登出成功");
    }
    
    /**
     * 从请求中提取Token
     */
    private String extractToken(HttpServletRequest request) {
        String authHeader = request.getHeader("Authorization");
        if (authHeader != null && authHeader.startsWith("Bearer ")) {
            return authHeader.substring(7);
        }
        return null;
    }
    
    /**
     * 计算Token剩余有效时间（秒）
     */
    private Long calculateTokenExpireSeconds(String token) {
        try {
            Claims claims = jwtUtil.parseToken(token);
            if (claims == null || claims.getExpiration() == null) {
                return null; // 使用默认TTL
            }
            
            long expireTime = claims.getExpiration().getTime();
            long currentTime = System.currentTimeMillis();
            long remainSeconds = (expireTime - currentTime) / 1000;
            
            // 如果已经过期，返回null（使用默认TTL）
            if (remainSeconds <= 0) {
                return null;
            }
            
            return remainSeconds;
        } catch (Exception e) {
            log.warn("[Logout] 计算Token过期时间失败", e);
            return null; // 使用默认TTL
        }
    }
}
```

## 🎯 六、实施建议

### 6.1 实施步骤

1. **第一步**：创建 `TokenBlacklistService`
2. **第二步**：修改 `JwtAuthFilter`，添加黑名单检查
3. **第三步**：创建 `AuthController`，实现登出接口
4. **第四步**：前端修改登出逻辑，调用后端接口
5. **第五步**：测试验证

### 6.2 注意事项

1. **Redis内存**：黑名单会占用Redis内存，但TTL会自动清理
2. **性能影响**：每次请求检查黑名单，Redis查询很快（<1ms），影响可忽略
3. **Token过期时间**：建议设置为7天，平衡安全性和用户体验
4. **异常处理**：即使后端登出失败，前端也要清除本地状态
